Layout

경험해 본 레이아웃 중 일부를 수정하여 공개합니다.
(디자인적 요소는 대부분 제거된 상태이며, 구조와 스타일 위주로 참고 부탁드립니다.)
작업물들은 git / webpack(build) 통해 이루어지고 있으며,PMS 툴로는 Wiki/Jira를 사용하고 있습니다.

해당 리스트는 비교적 최근 수행되었던 작업물이며, 신규 프로젝트는 scss를 사용하여 제작하고 있습니다.
최근 scss에 관심을 두고 실제 업무에 적용하며 배워가고 있습니다.

gulp를 사용하여 스프라이트 이미지를 제작한 경험이 있으며, 변수를 사용하여 서비스 내에서 사용될 공통 아이콘을 만들어서
스프라이트 이미지가 자동화 툴을 통해 수정되어도 position 수정이 필요 없도록 공통 작업을 했습니다.
디자인상의 이유로 매 프로젝트마다 요소 하나하나를 변수나 함수로 지정하여 적용하기는 어려우나,
주어진 환경 내에서 공통적 요소(뎁스별 제목, 버튼, 탭, 아이콘, 컬러, 반복적으로 사용되는 비슷한 리스트 등)를 만들어 좀더 효율적으로 스타일을 적용하기 위해 노력하고 있습니다.
